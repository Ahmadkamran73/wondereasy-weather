Component Relationships

App → Components
App component manages all child components and handles navigation between views
Search → Weather Service
City search components emit selected cities to trigger weather service calls
Weather Service → API
Service makes HTTP requests to OpenWeatherMap API with city parameters
Forecast → Display
Forecast component receives weather data and renders it in the UI
